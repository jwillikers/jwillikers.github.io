Unlock KeePassXC on Login

Jordan Williams included in Systems
 2020-09-19  90 words   One minute 
Contents

  1. Install KeePassXC.

    $ sudo snap install keepassxc

  2. Install the libsecret-tools package.

    $ sudo apt install libsecret-tools

  3. Create a secret

    $ secret-tool store --label="KeePass Passwords" keepass Passwords
Password:
    $ which keepassxc
