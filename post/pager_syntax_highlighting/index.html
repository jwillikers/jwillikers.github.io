Terminal Pager Syntax Highlighting

title = "Terminal Pager Syntax Highlighting" categories = ["Command-line"] tags = ["bat", "BSD", "delta", "Linux", "git", "less", "macOS", "source-highlight", "Ubuntu", "Unix", "Vim", "vimpager", "vimcat"] date = "2020-10-11" draft = true

The default pager in most Unix terminals, usually less, works great for a scrolling through text. It’s certainly a great improvement over its predecessor [more], though this doesn’t mean it can’t be improved.[1] One such area for enhancement is syntax highlighting, which can make scanning through code a bit less tedious.

Tutorial

There is more than one way to get pager syntax highlighting. The methods here include incorporating this functionality on top of less as well as switching to a different pager.

GNU Source-highlight

One popular choice is https://www.gnu.org/software/src-highlite/source-highlight.html [GNU Source-highlight], a general tool for applying source highlighting to a file.

delta

delta is a pager dedicated to providing syntax highlighting for git diffs.

Vim

Install Vim.

➜ sudo apt install -y vim

less.sh

/usr/share/vim/vim82/macros/less.sh

Create an alias for the less command which calls Vim’s less.sh script.

➜ alias -s less /usr/share/vim/vim81/macros/less.sh

This command uses the alias command provided by fish.

vimpager

Arch Wiki - Vim: Vim as a pager vimpager

vimpager and vimcat

  1. Grab the latest source code.

    ➜ git clone git://github.com/rkitover/vimpager
    Note:

    While packages may be available, the last release was five years ago depite the source code continuing to receive updates.[2]

  2. Install.

    # Debian / Ubuntu
➜ sudo make install-deb

# Everything else.
➜ sudo make install

  3. Register bat as an available pager for update-alternatives.

    ➜ sudo update-alternatives --install /usr/bin/pager pager /usr/bin/bat 0

  4. Choose bat as the default pager for update-alternatives.

    ➜ update-alternatives --set pager /usr/bin/bat

vimpager

  1. Use vimpager as the default pager.

    For systemd-homed users, use homectl to set the PAGER environment variable.

    ➜ homectl update jordan --setenv=PAGER=/usr/local/bin/vimpager

    Otherwise, use pam_env to set the PAGER environment variable.

    Otherwise, set the pager directly in your shell as shown here for fish.

    +

    ➜ set -Ux PAGER vimpager

  2. Configure git to use vimpager.

    ➜ git config --global core.pager /usr/local/bin/vimpager

  3. Use vimpager instead of less.

    ➜ alias -s less "$PAGER"

vimcat

Alternatively,

  1. Enable colored output in less.

    ➜ set -Ux LESS "-R"

  2. Imbue less with vimcat.

    ➜ set -Ux LESSOPEN "|vimcat -o - %s"
